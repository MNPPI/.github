# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any MNPPI project, please report it responsibly.

**Email:** mark@mnppi.org

Please include:

- A description of the vulnerability
- Steps to reproduce
- The affected project and version

I try to acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Scope

This policy applies to all public repositories under the [MNPPI](https://github.com/MNPPI) organization.
